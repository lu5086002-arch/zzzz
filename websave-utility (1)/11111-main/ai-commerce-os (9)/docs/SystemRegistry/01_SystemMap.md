## 3. 全局高频底层路由导航策略
- 输入 `"带我去商品库"` ➔ 智能映射打开：`ProductCenter`
- 输入 `"查看订单"` ➔ 智能映射打开：`OrderCenter`
- 输入 `"商家设置"` ➔ 智能映射打开：`EnterpriseSettings`
- 输入 `"财务中心"` ➔ 智能映射打开：`FinanceCenter`
- 输入 `"去AI大脑"` ➔ 智能映射打开：`EcosCEODashboard` (限 Super Admin 权)
- 输入 `"POS 机系统"` ➔ 智能映射：`Sales Channels ➔ Point Of Sale`
- 输入 `"CMS 管理"` ➔ 智能映射：`Publish Elements ➔ Content Management`
