# Data Dictionary: Phase 271-280 Fashion Demand Intelligence Engine

## 1. Domain Overview
The **Fashion Demand Intelligence Engine** provides Shopify-level multitenant macro and micro demand, trend, lifecycle, risk, elasticity, and autonomous decision oversight for Enterprise platform boards.

---

## 2. Table Definitions

### `demand_signal_sources`
Aggregates peripheral sensors, API scrapers, demographic metrics, and indexing channels.
- `id` (VARCHAR, PK): Unique source locator. (e.g. `dss_temp`).
- `name` (VARCHAR): Friendly descriptive name.
- `category` (VARCHAR): Enum constraint `(Weather, Temperature, Holiday, WageCycle, Tourism, SocialHeat, Competitor)`.
- `base_weight` (INT): Global credibility bias `0-100`.
- `is_active` (BOOLEAN): Status of crawler stream.

### `demand_signals`
Discrete signal instances captured by active crawlers.
- `id` (VARCHAR, PK): Signal locator.
- `source_id` (VARCHAR, FK): Points to parent source.
- `signal_type` (VARCHAR): Detailed signal spec. Like "Severe Alpine Frost (-6°C)".
- `magnitude_score` (INT): Measured intensity `0-100`.
- `recorded_at` (TIMESTAMP): Event recording instant.
- `status` (VARCHAR): Process status: `Active`, `Processed`, `Suppressed`.

### `regional_forecasts_v2`
Spatially-discrete demographic product classification predictions.
- `id` (VARCHAR, PK)
- `country` (VARCHAR): ISO standard 2-digit code (e.g., `FR`, `DE`, `GB`).
- `category_name` (VARCHAR): Categorical domain.
- `time_horizon` (VARCHAR): Predicted time scope `("7d", "30d", "90d")`.
- `forecasted_growth_pct` (NUMERIC): Growth expectation percentage.
- `confidence_score` (INT): Engine evaluation metric `0-100`.

### `trend_signals_v2`
Social media sentiment and runway aesthetic signals.
- `id` (VARCHAR, PK)
- `trend_name` (VARCHAR): Runway name or category tag (e.g. `Quiet Luxury`).
- `signal_strength` (INT): Normalised visibility weight `0-100`.
- `source_platform` (VARCHAR): Scraped source stream name.

### `price_elasticity_models`
Identifies consumer product response sensitivity ratios.
- `id` (VARCHAR, PK)
- `product_id` (VARCHAR): Associated SKU locator.
- `elasticity_coefficient` (NUMERIC): Response ratio (volume vs delta price). Negative values represent classic downward sloping demand.
- `optimal_price` (NUMERIC): Target revenue-maximizer price.
- `current_price` (NUMERIC): Present catalog base price.

### `forecast_board_decisions`
Autonomous governance actions generated by board models.
- `id` (VARCHAR, PK)
- `report_id` (VARCHAR, FK): Source report context.
- `subject` (VARCHAR): Topic description.
- `required_action` (TEXT): Algorithmic operational suggestion.
- `p_success` (NUMERIC): Likelihood of successful fulfillment `0-100`.
- `status` (VARCHAR): State tracker `(Pending, Approved, Rejected)`.
