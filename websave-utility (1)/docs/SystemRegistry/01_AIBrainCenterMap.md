# AI 大脑中心全地图 (AI Brain Center Map)

## 📌 大脑中心宪章
本中心是 **ECOS Platform (Super Admin)** 的最高智能控制中心。依据隔离红线，此处的深度推理链、贝叶斯反思图谱及多智脑博弈面板，**绝对隐藏**于普通商家管理界面。

---

## 🧠 核心 10 大脑部组件与功能矩阵

### 1. `dashboard` (总裁今日经营大盘 / CEO Dashboard)
* **对应文件路径**: `src/components/admin/ai-brain-center/EcosCEODashboard.tsx`
* **功能描述**: 今日销售业绩、商户高利润机会预测、法国市场 ROI 预警。不显示无聊的假 AI 智商指数，直接展示“今日赚多少”、“哪里面临亏损”。
* **完成状态**: `[LIVE]`
* **核心内容**: 销售利润宏观洞察、动态 AI 建议大卡。

### 2. `execution_center` (决策执行控制中心 / Execution Center)
* **对应文件路径**: `src/components/admin/ai-brain-center/AIExecutionControlCenter.tsx`
* **功能描述**: 自动审查 AI 发起的行动（如自动对齐货源、营销发放），提供拒绝、批准、重试的热通道，保障系统自愈对空安全。
* **完成状态**: `[LIVE]`
* **核心内容**: 审批审批流、手动挂起。

### 3. `discovery` (自动盲点感知探索引擎 / AI Discovery)
* **对应文件路径**: `src/components/admin/ai-brain-center/AIDiscoveryCenter.tsx`
* **功能描述**: 扫描多门店历史销售，自动找到高可信度高毛利爆款（如“羊毛夹克在法国转化率上涨22%”）并计算复合预测分阶。
* **完成状态**: `[LIVE]`
* **核心内容**: 蓝海品类发现预测、热力学因果盲区图谱。

### 4. `monitoring` (大盘边缘智能监控 / Monitor Engine)
* **对应文件路径**: `src/components/admin/ai-brain-center/PlatformIntelligenceCenter.tsx`
* **功能描述**: 收集各租户在执行端产生的性能损耗、API 网关劫持率、系统算效延迟等底层状态。
* **完成状态**: `[LIVE]`
* **核心内容**: 边缘延迟图表、CPU 与并发流量波谱。

### 5. `optimizer` (企业算效最大化调试器 / Token Optimizer)
* **对应文件路径**: `src/components/admin/ai-brain-center/EcosPerformanceOptimizer.tsx`
* **功能描述**: 监控大语言模型 API 投掷频率，并拦截高冗余的无意识反问，压缩 Token 成本。
* **完成状态**: `[LIVE]`
* **核心内容**: Token 压缩率指示波、缓存拦截参数。

### 6. `strategic` (长程商战防御模拟沙盘 / Strategic Engine)
* **对应文件路径**: `src/components/admin/ai-brain-center/EcosStrategicIntelligence.tsx`
* **功能描述**: 模拟跨国供应链竞争及海外低价降维战役，建立竞争博弈矩阵，帮助平台商家指定毛利安全红线。
* **完成状态**: `[LIVE]`
* **核心内容**: 逆回因反事实分析树、收益损耗博弈树。

### 7. `cognitive` (最高认知宪章审计委员会 / Cognitive Governance)
* **对应文件路径**: `src/components/admin/ai-brain-center/EcosCognitiveGovernance.tsx`
* **功能描述**: 用于审查 AI 是否在法律、税法、品牌主张和店主毛利率底线下做出决策（例如：检查 AI 是否偷偷把折扣放宽至 20% 以下破坏毛利率底线）。
* **完成状态**: `[LIVE]`
* **核心内容**: 审计日志序列、自愈防火墙阈值。

### 8. `nervous` (企业高频突触大盘 / Enterprise Nervous System)
* **对应文件路径**: `src/components/admin/ai-brain-center/EcosEnterpriseNervousSystem.tsx`
* **功能描述**: 检测多租户分布式底层数据状态的呼吸变频波，秒级输出高危库同步阻塞警报。
* **完成状态**: `[LIVE]`
* **核心内容**: 数据突触呼吸线图。

### 9. `memory` (长期事实与规则记忆中枢 / Memory Console)
* **对应文件路径**: `src/components/admin/ai-brain-center/EcosGovernanceConsole.tsx`
* **功能描述**: 维护企业“DNA 级记忆库”（如：第一核心市场 = 法国，品牌主张 = 中高端，毛利率底线 = 35%），确保 AI 的商业决策始终契合基础定位。
* **完成状态**: `[LIVE]`
* **核心内容**: 可视化规则编辑器、记忆块持久化状态。

### 10. `boardroom` (多智能体辩论董事会 / AI Boardroom)
* **对应文件路径**: `src/components/admin/ai-brain-center/EcosAdvancedLayers.tsx`
* **功能描述**: 渲染“财务脑、库存脑、营销脑、风控脑”的异议博弈辩论：营销脑主张加大广告投放，财务脑因现金流预警驳回，最终 CEO 智脑通过“先补货后打广告”中立自愈方案。
* **完成状态**: `[LIVE]`
* **核心内容**: 实时智脑辩论对话框、决议投票卡。
